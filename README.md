# Portfolio Tech

Raccolta di piccoli progetti tech realizzati da [Luigi Garone](https://github.com/LuigiGarone).

## Progetti

### [Simulazione Scioglimento Ghiacci](Simulazione-Scioglimento-Ghiacci/)
Simulazione interattiva dell'innalzamento dei mari basata su dati IPCC, NASA e NOAA. Slider 0-65m, mappa mondiale e dettaglio Italia con scenari preimpostati.

**[Prova la demo live](https://luigigarone.github.io/Portfolio-Tech/Simulazione-Scioglimento-Ghiacci/innalzamento-dei-mari.html)**

### [Estrazione Vincitore Pasqua](Estrazione-Vincitore-Pasqua/)
App desktop per l'estrazione casuale di un vincitore da un elenco di partecipanti (Excel/CSV). Countdown visivo e interfaccia intuitiva.

### [Videogioco ELIA (NeonBreak)](Videogioco-ELIA/)
Arkanoid in stile cyberpunk con 5 livelli, power-ups e tracking degli high score. HTML5/Canvas.

## Tech Stack
- HTML5, CSS3, JavaScript vanilla
- D3.js, TopoJSON, Leaflet (simulazione ghiacci)
- Electron (estrazione vincitore)
- Canvas API (videogioco)
